The Code for TacoCat


                        // controller function
                        function getVals() {
                            document.getElementById("alert").classList.add("invisible");

                            let userInput = document.getElementById("userInput").value;
                            let returnObj = checkPalin(userInput);
                            display(returnObj);
                        }

                        // logic function
                        function checkPalin(string) {
                            string = string.toLowerCase();
                            let regex = /[^a-z0-9]/gi;
                            string = string.replace(regex,"");

                            let revString = [];
                            let returnObj = {};

                            for (let i = string.length - 1; i >= 0; i--) {
                                revString += string[i];
                            }

                            if (revString == string) {
                                returnObj.msg = "Well Done! You entered a palindrome!";
                            } else {
                                returnObj.msg = "Sorry! You did not enter a palindrome!";
                            }

                            returnObj.reversed = revString;
                            return returnObj;
                        }

                        // view function
                        function display(returnObj) {
                            document.getElementById("alertHeader").innerHTML = returnObj.msg;
                            document.getElementById("msg").innerHTML = `Your phrase reversed is: ${returnObj.reversed}`;
                            document.getElementById("alert").classList.remove("invisible");
                        }

The Code is structured in three functions:

getVals()

The function getVals() is the controller function. It starts by getting the id of alert to hide the alert so it only shows after a user has entered their word. By using the id element method, we can get the string that the user entered in. Then, another object is created to store the result of the second function checkPalin(), which returns an object saying if the string entered is a palindrome or not. That object is used as the input for the display function, which displays the results back to the user.

checkPalin()

The function checkPalin() is the logic function, and takes in a string parameter. Before we can check whether the string is a palindrome, any spaces, capital letters, or other punctuation marks need to be removed. This is done with .toLowerCase() and regex, and the /[^a-z0-9]/ means that only letters a through z and letters 0 through 9 are allowed; anything otherwise is taken out. After, an array revString and an object returnObj are initialized. A for loop is used to reverse the user-entered string, using a starting index of the string's last character and making its way to the front. Through each iteration of the loop, the letter of the string is being added to the empty array revString, so by the end there is a fully reversed string. Since a palindrome reads the same forwards and backwards, a simple if/else statement can check if the string is a palindrome. If the reversed string is the same as the user-entered string, then it is a palindrome and the returnObj is set with a message responding accordingly to the result. Then the reversed version of the input is stored in the returnObj, which is later used in display(). returnObj is the output of checkPalin().

display()

The function display() is the view function and takes a parameter returnObj. To get to the alert section of the HTML, document.getElementById() is used, and the message that was returned in checkPalin() is input into the HTML with innerHTML. The user is displayed whether their entered string is a palindrome or not, and the reversed string. The "invisible" class that hid the alert is removed so the user can see their final result.
